Django Project Template


Welcome!! Thank you for trying this Django template from NotRandom.Net. This is a template of a Django project that, I hope, will help people that want to learn how to use Django with a basic example of a Django project. The project includes several features:

  • Docker-ready Run the project from a docker container using a PostgreSQL database. In the 'volumes' folder you will find the container files, stored in your (host) computer. Take a look at the 'docker' folder and Test docker and docker compose. Don't forget to setup the .env file as 'local' to run a test container in your PC.
  • Multiple Deployment Modes: The project is enabled to work in different envrionment modes depending of the type of deployment you need to do. If you want to start with tests in your local PC, set the '.env' file to 'local_wo_container' (Local Without Container, using the included SQLite3 database). Or pick 'dev' or 'production' if you want to use with docker in other environments.
  • Nginx sample Included uncomment the Nginx service in the docker/docker-compose.yml file and get some ideas on how to use the popular Nginx webserver with your django project in a docker container environment.
  • Working CSS: Making the CSS and static content work can be a bit frustrating, as there are severa; factors to keep in mind. This working example can help you to review how to set it up.
  • Bootstrap 5: The 'base.html' file provides a working example of how to setup bootstrap. Take a look at the Bootstrap Page and see all the cool things you can do with it + Django.


Enjoy and I hope you find it useful!!
